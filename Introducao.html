O que é o ReactJs

  • É uma biblioteca de JavaScript.
  • Utilizado para criar interfaces de usuários (UI).
  • Pode ser usada para criar SPA (pacotes extra).
  • Baseado em uma filosofia de layout por componentes.
  • permite a criação de aplicações mobile React Native.

Qual a diferença entre React e Angular

Angular

Angular é uma estrutura JavaScript para criar aplicativos na Web, desktop e até aplicativos móveis nativos. Uma das principais diferenças entre angular e reação é que o Angular é de fato um kit de ferramentas completo para criar seus aplicativos. Fora da caixa, a angular possui soluções para roteamento, acesso a serviços de dados, modelos e muito mais. Com o Angular, os usuários têm tudo o que devem usar, proporcionando muita consistência. O Angular permite que o usuário decida como estruturar seu projeto para que o código seja facilmente encontrado e mantido. A estrutura existe há muito tempo e possui uma grande comunidade de desenvolvedores de código aberto.

React ou ReactJs

O React.js, geralmente chamado de React or ReactJS, é uma biblioteca JavaScript responsável pela construção de vários componentes da interface do usuário ou, em outras palavras, responsável pelas variantes dos componentes da interface do usuário. Ele fornece suporte para frontend e servidor. O reação é desenvolvido pelo Facebook. É mais comumente usado para lidar com a camada de visualização de aplicativos da Web e móveis. O ReactJS / React permite que os usuários criem componentes reutilizáveis da interface do usuário. Atualmente, é uma das bibliotecas JavaScript mais populares e possui uma base sólida e uma grande comunidade por trás.

Site do React

Declarativo

React faz com que a criação de UIs interativas seja uma tarefa fácil. Crie views simples para cada estado na sua aplicação, e o React irá atualizar e renderizar de forma eficiente apenas os componentes necessários na medida em que os dados mudam. Views declarativas fazem com que seu código seja mais previsível e simples de depurar.

Baseado em componentes

Crie componentes encapsulados que gerenciam seu próprio estado e então, combine-os para criar UIs complexas. Como a lógica do componente é escrita em JavaScript e não em templates, você pode facilmente passar diversos tipos de dados ao longo da sua aplicação e ainda manter o estado fora do DOM.
